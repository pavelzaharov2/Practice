Задание 3. 
Если введённое число не делится ни на 5, ни на 7, вывести 00. Если 
введённое число делится на 5, но не на 7, вывести 01. Если введённое 
число делится на 7, но не на 5, вывести 10. Если введённое число 
делится и на 5, и на 7, вывести 11.


print('введите число')
A=int(input())
if A%5 == 0 and A % 7 == 0: print('11')
elif A%5 == 0 and A % 7 != 0: print('01')
elif A%5 != 0 and A % 7 == 0: print('10')
else: print('00')
